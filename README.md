# sliding-clock
Essay on a new kind of design clock

## Requirements
- uv for package management (`pip install uv`)
- python 3.13+ (`python3 --version`)

## How to run it?
1. `git clone https://github.com/ostaquet/sliding-clock.git` to get the repository
2. `cd sliding-clock` to go into the downloaded source code
3. `uv venv` to create the virtual environment
4. `source .venv/bin/activate` to activate the virtual environment
5. `uv pip install -r ./requirements.txt` to install the dependencies
6. `python main.py` to enjoy
